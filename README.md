# joboard.
Find out where there are good job prospects in Denmark. 

## About
[Arbejdsmarkedsbalancen.dk](https://arbejdsmarkedsbalancen.dk/) is tough to navigate if you don't already know Danish. Trying to build a job search tool that's more newcomer-friendly.
